CLI capability map

One command family, one proof surface

StoreConfigure PostgreSQL, MySQL, or SQLite compatibility Stores and keep runtime facts in SQL.
EnvironmentRegister targets, inspect bootstrap plans, restore runtimes, and publish verified environments.
CasesDiscover, prioritize, run, and inspect HTTP API cases with request and response Evidence.
WorkflowRun ordered API workflows and keep per-step timing, status, logs, and topology context.
EvidenceReview request, response, assertion, timing, log, artifact, and topology records by run id.
ReportsGenerate compact JSON and HTML output for agents, CI, and human reviewers.
